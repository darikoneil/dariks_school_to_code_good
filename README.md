# dariks_school_to_code_good

# Example project for ruff, pytest, & sphinx

This repository contains a small example module you can run tools on:
- example_code/example_src.py

Quick commands:
- Update virtual environment: uv sync
- Format: ruff format --fix .
- Lint: ruff check .
- Tes: pytest

See docs/ruff.md for more details.
